# Retail Store Sales Dashboard (Excel Project)

## Project Overview

This project presents a Retail Store Sales Dashboard developed using Microsoft Excel to analyze store performance and sales trends. The dashboard helps identify top-performing products, sales contributors, and regional performance insights.

## Tools Used

- Microsoft Excel

## Key Features

- Product-wise sales analysis
- Salesperson performance tracking
- Region-wise sales comparison
- Monthly trend analysis
- KPI summary for total sales and units sold
